# UniVault Development Log

## Project Overview

**Project Name:** UniVault
**Description:**
UniVault is a university community platform where students can share and access academic resources such as past exams, TPC solutions, lecture notes, and important study materials.

**Main Goal:**
Create a responsive web platform that improves collaboration and knowledge sharing among university students.
